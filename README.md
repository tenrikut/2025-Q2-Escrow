## About

The **2025-Q2-Escrow** project is an escrow service designed to securely manage funds between multiple parties. The system ensures that all agreed-upon conditions are met before releasing the funds, providing a robust and transparent solution for financial transactions.

---

## Features

- **Secure Transactions**: Funds are held securely until all conditions are fulfilled.
- **Multi-Party Support**: Supports transactions involving multiple participants.
- **Smart Contract Integration**: Implements business logic using Rust-based smart contracts.
- **TypeScript Frontend**: A user-friendly interface for interacting with the escrow system.
- **High Performance**: Built with Rust for speed and reliability.

## Getting Started

To get started with this project, follow the steps below:

### Prerequisites

- Install [Rust](https://www.rust-lang.org/tools/install).
- Install [Node.js](https://nodejs.org/) (for TypeScript development).
- Install a package manager like [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tenrikut/2025-Q2-Escrow.git
   cd 2025-Q2-Escrow
   ```

2. Build the Rust backend:
   ```bash
   cd backend
   cargo build
   ```

3. Set up the TypeScript frontend:
   ```bash
   cd frontend
   npm install
   npm run start
   ```

4. Run the full application:
   - Backend and frontend details to be provided as implementation progresses.
